# My Dotfiles

My personal Linux dotfiles managed with GNU Stow.

## Repository Structure

```
bash/
git/
profile/
vscode/
fastfetch/
btop/
gtk/
scripts/
```

## Requirements

- Git
- GNU Stow

## Clone

```bash
git clone https://github.com/YOUR_USERNAME/dotfiles.git ~/dotfiles
```

## Apply Dotfiles

```bash
cd ~/dotfiles

stow bash
stow git
stow profile
```
